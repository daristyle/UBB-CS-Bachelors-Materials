=================    TASK 1    ====================
Implement the following task using 2 ASM submodules and 1 main C/C++ module.
You may copy & paste this task inside your source code if it makes it easier for you to implement it.

Write the main.c/main.cpp where you read the following:
* on the first line, you read N - a number greater than 0
* on the next N lines, you read s[i] - a string made up of letters, a-z, A-Z, and digits, 0-9.

For each s[i], sum the digits inside the word (2.5p):
* must be implemented inside sum_digits.asm
* if there are no digits, return 0
* suggestion: implement sum_digits(const char* string)

If the number of vowels is a perfect square, add it to the total sum (1.5p):
* must be implemented inside perfect_square.asm
* suggestion: implement an is_perfect_square function inside the ASM module
* if you skip checking for the number to be a perfect square, just add it to the total sum.

Write the total sum in a file called sum_of_squares.txt.

=================   EXAMPLES   =====================
>>> INPUT 1 (console):
    1
    a1b2c -> 3 (not perfect square)
>>> OUTPUT 1 (sum_of_squares.txt):
    0

>>> INPUT 2 (console):
    3
    abc123 -> 6 (not perfect square)
    test4 -> 4 (perfect square: 2*2)
    hello -> 0 (perfect square: 0*0)
>>> OUTPUT 2 (sum_of_squares.txt):
    4

>>> INPUT 3 (console):
    4
    a1b2c1 -> 4 (perfect square: 2*2)
    x9y -> 9 (perfect square: 3*3)
    Code2025 -> 9 (perfect square: 3*3)
    noDigits -> 0 (perfect square: 0*0)
>>> OUTPUT 3 (sum_of_squares.txt):
    22
