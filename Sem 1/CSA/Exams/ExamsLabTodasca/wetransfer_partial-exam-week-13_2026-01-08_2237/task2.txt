=================    TASK 1    ====================
Implement the following task using 2 ASM submodules and 1 main C/C++ module.
You may copy & paste this task inside your source code if it makes it easier for you to implement it.

Write the main.c/main.cpp where you read the following:
* on the first line, you read N - a number greater than 0
* on the next N lines, you read a[i] - a number between 0 and 10^6

For each a[i], compute the reverse number (2.5p):
* must be implemented inside reverse_number.asm
* discard any 0 digits from the end
* suggestion: implement int reverse_number(int num)
* reverse of 123  -> 321
* reverse of 1205 -> 5021
* reverse of 3400 -> 43

Add the square root of each reverse number to the total sum (1.5p):
* must be implemented inside square_root.asm
* suggestion: implement an int square_root function inside the ASM module
* if you skip checking computing the square root, just add it to the total sum.
* we will obviously NOT use any rational numbers
* square root of 49 -> 7
* square root of 52 -> 7
* square root of 64 -> 8

Write the total sum in a file called sum_of_roots.txt.

=================   EXAMPLES   =====================
>>> INPUT 1 (console):
    1
    49 -> reverse = 94 -> sqrt(94) = 9
>>> OUTPUT 1 (sum_of_roots.txt):
    9

>>> INPUT 2 (console):
    3
    123 -> reverse = 321 -> sqrt(321) = 17
    100 -> reverse = 1 -> sqrt(1) = 1
    64 -> reverse = 46 -> sqrt(46) = 6
>>> OUTPUT 2 (sum_of_roots.txt):
    24

>>> INPUT 3 (console):
    4
    1205 -> reverse = 5021 -> sqrt(5021) = 70
    3400 -> reverse = 43 -> sqrt(43) = 6
    49 -> reverse = 94 -> sqrt(94) = 9
    100 -> reverse = 1 -> sqrt(1) = 1
>>> OUTPUT 3 (sum_of_roots.txt):
    86
