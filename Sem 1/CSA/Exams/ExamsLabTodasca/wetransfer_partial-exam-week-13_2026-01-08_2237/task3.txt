=================    TASK 1    ====================
Implement the following task using 2 ASM submodules and 1 main C/C++ module.
You may copy & paste this task inside your source code if it makes it easier for you to implement it.

Write the main.c/main.cpp where you read the following:
* on the first line, you read N - a number greater than 0
* on the next N lines, you read s[i] - a string made up of letters, a-z, A-Z.

For each s[i], count the number of vowels inside the word (aeiou and AEIOU) (2.5p):
* must be implemented inside count_vowels.asm
* suggestion: implement count_vowels(const char* string) OR count_character(const char* string, character)
    and call count_character(string, 'a'), count_character(string, 'e') ...

If the number of vowels is a prime number, add it to the total sum (1.5p):
* must be implemented inside prime.asm
* suggestion: implement an is_prime function inside the ASM module
* if you skip checking for the number to be a prime number, just add it to the total sum.

Write the total sum in a file called sum_of_vowels.txt.

=================   EXAMPLES   =====================
>>> INPUT 1 (console):
    1 -> 0
    aeixx -> 3
>>> OUTPUT 2 (sum_of_words.txt):
    3

>>> INPUT 2 (console):
    3
    hello -> 2
    world -> 1 (not prime)
    test -> 1 (not prime)
>>> OUTPUT 3 (sum_of_words.txt):
    2

>>> INPUT 3 (console):
    4
    AEiou -> 5 (prime)
    programming -> 3 (prime)
    xyz -> 0
    Education -> 5 (prime)
>>> OUTPUT 3 (sum_of_words.txt):
    13
